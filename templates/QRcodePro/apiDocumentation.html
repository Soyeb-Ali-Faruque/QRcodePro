Text Service

Generate a QR code for plain text.

Note: QR code customization is optional. If a logo is provided, it should be sent as a base64-encoded string. You can choose the logo shape (Circle or Square) and corner style (Sharp or Rounded).

Method: POST

Request

{
    #URL for sending Request
    https://soyebalifaruque.pythonanywhere.com/qrcodepro/api/generate-qr/


    "type": "text",
    "text_content": "Hello, World!",
    "fill_color": "#000000",
    "background_color": "#FFFFFF",
    "logo": "base64encodedLogo",   // Optional: Provide base64-encoded logo if available
    "logo_shape": "circle",        // Optional: 'circle' or 'square'
    "logo_corner": "sharp"         // Optional: 'sharp' or 'rounded'
}

Response

{
    "qr_code": "base64encodedQR"
    "status":"success or fail"
}
